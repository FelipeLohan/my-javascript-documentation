# OPERADORES

## Operadores aritméticos:

```
- + (soma)
- - (subtração)
- * (multiplicação)
- / (divisão)
- % (mod / resto da divisão)
- ** (exponenciação)
```

## Operadores comparativos:

```
- < (menor que)
- > (maior que)
- <= (menor ou igual que)
- >= (maior ou igual que)

- == (compara o valor)
- === (compara o tipo e valor)
```

## Operadores lógicos:

```
- && (e)
- || (ou)
- ! (negação)
```

#### OBS:

Operadores comparativos e lógicos sempre retornam ` true ` ou ` false `.
